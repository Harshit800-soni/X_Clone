# X Clone 🐦

A responsive X (formerly Twitter) Clone built using HTML, CSS, Tailwind CSS, and JavaScript. This project recreates the modern user interface of X with a clean design and responsive layout for different screen sizes.

## 🚀 Features

- Responsive X-inspired UI
- Modern and clean design
- Sidebar navigation
- Feed/Timeline section
- Trending section
- User profile area
- Mobile-friendly layout
- Built with Tailwind CSS

## 🛠️ Tech Stack

- HTML5
- CSS3
- Tailwind CSS
- JavaScript
- Node.js & NPM

## ⚙️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/x-clone.git
cd x-clone
```

### Install Dependencies

```bash
npm install
```

### Run Tailwind CSS

```bash
npm run dev
```

### Open the Project

Open `index.html` in your browser or use Live Server in VS Code.
